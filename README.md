# World Cup Oracle
Predicts World Cup match outcomes (scikit-learn), explains them with a Claude agent
analyst, and wraps it in an interactive experience. UC Berkeley AI Hackathon 2026.

## Setup
    python3 -m venv .venv && source .venv/bin/activate
    pip install -r requirements.txt
    cp .env.example .env      # then fill in your keys
